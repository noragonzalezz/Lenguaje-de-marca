HTML

HTML son las siglas de HyperText Mrkup Language , que puede traducirse como lenguaje de marcas o marcado de hipertexto.


El lenguaje HTML, se emplea para crear las paginas web.Es muy facil ver el codigo HTML de una pagina web, la opcion exacta cambia de un navegador a otro y también puede cambiar de una versión a otra de un mismo navegador, pero suelen ser un nombre similar.

HTML se compone de etiquetas que se escriben entre los simbolos menor y mayor que.
